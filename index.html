    Bienvenido a Lanzo POS

    Lanzo es un sistema completo de punto de venta y gestión para pequeños negocios con las siguientes secciones principales:

    • Punto de Venta: Interfaz para tomar pedidos, calcular totales y procesar ventas.
    • Gestión de Productos: Permite añadir, editar y eliminar productos.
    • Panel de Ventas: Muestra estadísticas e informes de ventas.
    • Configuración: Personaliza datos de la empresa y aspecto visual.

    Características: diseño responsive para celulares, tablets y computadoras, cálculo de costos, persistencia de datos local y mucho más.
